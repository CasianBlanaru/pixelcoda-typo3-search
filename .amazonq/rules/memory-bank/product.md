# Product Overview

## Purpose
TYPO3 Headless CMS with Next.js frontend - a modern decoupled content management system combining TYPO3 14 backend with React-based frontend rendering. Enables content editors to manage content through TYPO3 while delivering modern web experiences through Next.js.

## Key Features
- **TYPO3 14 Headless Backend**: JSON API-driven content delivery via friendsoftypo3/headless extension
- **Next.js 16 Frontend**: React 19-based SSR/SSG rendering with @pixelcoda/headless-nextjs integration
- **Frontend Editing**: In-place content editing with pixelcoda/fe-editor - inline text editing, drag-and-drop, AI writing assistant
- **GSAP Animations**: Scroll-triggered animations through pixelcoda/content-gsap-animation extension
- **Search Module**: Custom TYPO3 search functionality via pixelcoda/typo3-search
- **Railway Deployment**: Production-ready Docker configuration with automated setup scripts
- **DDEV Development**: Local development environment with full TYPO3 and Next.js support

## Target Users
- Content editors needing visual frontend editing without backend complexity
- TYPO3 developers building modern headless architectures
- Development teams requiring decoupled CMS with React frontend
- Organizations deploying TYPO3 with cloud-native infrastructure

## Use Cases
- Corporate websites with content managed in TYPO3 rendered through Next.js
- Marketing sites requiring frequent content updates with frontend editing
- Multi-channel content delivery from single TYPO3 backend
- Headless CMS with AI-assisted content creation and inline editing
